# Paid Tools Approval Template

Before using paid tools or credits, the agent must ask:

```md
## Paid Tool Approval Needed

Tool:
Purpose:
Why free options are insufficient:
Estimated cost/credit usage:
Data/assets that will be sent:
Output expected:
Risk:
Free alternative:
Exact approval requested:
```

No purchase, subscription, credit usage, or paid API call should happen before approval.
