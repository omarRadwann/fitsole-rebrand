# 64 — Awwwards Jury and Scoring Master

What the Awwwards jury actually scores, how the math works, and where AI-built sites lose points. Use this when filling `docs/awwwards-jury-scorecard.md`.

## The four official categories

Awwwards rates entries across four pillars, each on a 1–10 scale (with one decimal):

1. **Design** (40% weight) — visual quality, typography, color, composition, hierarchy, originality.
2. **Usability** (30% weight) — navigation clarity, information architecture, accessibility, responsiveness, error states.
3. **Creativity** (20% weight) — distinctiveness, concept strength, signature interaction, courage of decisions.
4. **Content** (10% weight) — quality of copy, imagery, video, and the way the content carries the idea.

The final score is the weighted average. Honorable Mention sits around 6.5; Site of the Day generally requires 6.5+ with strong individual scores; Developer Award and Mobile Excellence have specific bars.

## Reading the four scores like a jury

### Design (the visual)

Jury looks for:
- typographic confidence and rhythm
- composition that isn't a stack of centered sections
- color used as an idea, not as decoration
- spacing system that feels intentional at every scale
- one or two brave decisions
- detail consistency (radius, shadow, icon, motion)

Jury penalizes:
- defaults from popular UI libraries used as the visual identity
- gradient blobs as personality
- mismatched icon sets
- random radii / shadow values
- screenshots that could have been generated by anyone

### Usability (the experience)

Jury looks for:
- 5-second clarity on what the site is
- obvious primary action
- working keyboard navigation
- focus states that match the visual system
- mobile composition that is its own design, not a squeeze
- forms that explain errors well

Jury penalizes:
- scroll-jacking with no purpose
- hidden navigation without a strong reason
- contrast failures
- mobile menus that hide essential content
- broken back button / browser history

### Creativity (the idea)

Jury looks for:
- one signature thing (idea, visual, interaction, screenshot)
- restraint as a deliberate choice
- a concept tied to the business, not to a trend
- bravery — at least one decision that could have been wrong

Jury penalizes:
- "AI-fancy" — many effects, no idea
- generic "premium" with no point of view
- copying a recently-famous site
- 3D applied where 2D would have been stronger

### Content (the words and pictures)

Jury looks for:
- copy that could only be this business
- imagery that is sourced, generated, or produced with intent
- a content tone that matches the visual tone
- proof that backs claims

Jury penalizes:
- buzzword copy
- stock imagery used as content
- inconsistent voice
- fake metrics or testimonials

## Mobile Excellence and Developer Award

These are awarded separately and have their own bars:

- **Mobile Excellence** — the site must work on mobile as a designed experience, not a fallback. Touch targets, gestures, viewport composition, image weight, and motion are scrutinized.
- **Developer Award** — code quality, performance, accessibility, novel technical solutions, and a credible repo or technical write-up are scrutinized.

A site can win SOTD without these; a site can win one of these without SOTD.

## Threshold targets

| Score | Status |
|---|---|
| < 6.0 | Below jury bar; treat as draft |
| 6.0–6.4 | Honorable Mention territory |
| 6.5–7.5 | SOTD plausible if uniqueness is high |
| 7.5+ | SOTY consideration |

These are public Awwwards bands; treat them as orientation, not a guarantee.

## Where AI builds usually lose

In order of frequency:

1. **Creativity 5–6** — competent execution, no idea.
2. **Content 5–6** — buzzwords; stock imagery.
3. **Design 6–7** — clean but generic; default UI library feel.
4. **Usability 7–8** — usually fine because the AI built it conservatively.

The leverage points are Creativity and Content. A site that fixes these two without losing Design/Usability climbs from 6.0 to 7.0+.

## The honest self-jury rule

Score yourself harshly. If your average is above 8.0 in self-jury, you're probably wrong about at least one category. Ask: which score would the actual jury cut by 1.5 points and why? Fix that.
