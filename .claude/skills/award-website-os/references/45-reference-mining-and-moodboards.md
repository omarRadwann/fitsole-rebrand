# Reference Mining and Moodboard Guide

Claude Code should not copy references. Claude Code should extract patterns, tension, and missed opportunities. This file tells Claude Code how to use references without producing derivative work.

## Reference categories

For each real project, collect or infer references in six buckets:

1. Category competitors: what buyers already expect.
2. Category winners: the best version of the obvious answer.
3. Out-of-category inspiration: museums, games, fashion, architecture, film, luxury, industrial design.
4. Typography references: type scale, pairing, editorial rhythm.
5. Motion references: pace, easing, reveal style, interaction behavior.
6. Technical references: Three.js, R3F, Spline, shader, GLB, performance examples.

## How to mine a reference

For every reference, capture:

- URL or source.
- What emotion it creates.
- What design device creates that emotion.
- What should be borrowed as a principle.
- What must not be copied.
- Performance or accessibility risk.
- How it could translate to mobile.

## Reference extraction table

| Reference | Bucket | Useful principle | Avoid copying | Mobile lesson | Risk |
|---|---|---|---|---|---|
| Add source | Category | Clear proof pattern | Visual skin | Collapse hierarchy | Too familiar |
| Add source | Out-of-category | Material/atmosphere | Direct layout | Simplify image crop | Heavy assets |
| Add source | Motion | Easing/reveal logic | Exact animation | Reduce motion state | CPU/GPU cost |

## High-value reference sources

Use sources like Awwwards, The FWA, Siteinspire, Godly, Land-book, One Page Love, Typewolf, Fonts In Use, Brand New, Mobbin, Lapa Ninja, Three.js examples, pmndrs examples, Spline examples, GSAP demos, and official framework docs. Confirm licenses before using assets.

## Moodboard synthesis

Do not create a moodboard as a collage only. Convert it into rules:

- Type: chosen direction and why.
- Color: atmosphere and contrast logic.
- Layout: grid, density, and rhythm.
- Imagery: photography, generated visuals, 3D, texture, or illustration.
- Motion: speed, easing, scroll behavior, and interaction level.
- Proof: how credibility appears visually.

## Anti-copy rule

If a reference strongly inspires the design, Claude Code must transform at least three dimensions: industry context, layout structure, asset system, motion behavior, color world, or copy voice. Never recreate a distinctive site's exact composition.
