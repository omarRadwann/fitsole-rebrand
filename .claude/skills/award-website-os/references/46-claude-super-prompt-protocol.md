# 46 — Claude Code Super Prompt Protocol

Use prompts that force Claude Code to work like a studio, not a single generic builder.

## Master Pattern

```txt
Use the award-website-os skill. Read SOURCE_OF_TRUTH.md, AGENTS.md, START_HERE.md, and SKILL.md. Business type: [type]. Spawn the required Claude Code agents, wait for reports, fill evidence docs, create three concepts, select one, define art direction, build, screenshot, critique, repair, run checks, and truth-label the final ship decision.
```

## Design-Specific Pattern

```txt
Before coding, define the signature idea, signature visual, signature interaction, type system, color system, asset strategy, motion language, mobile hero, and anti-cliche list. Reject generic SaaS layouts and meaningless 3D.
```
