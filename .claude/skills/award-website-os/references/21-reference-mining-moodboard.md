# 21 — Reference Mining and Moodboard Intelligence

## Mission

Use references intelligently without copying.

## Reference Types

Collect:
- competitor references
- category references
- visual mood references
- interaction references
- typography references
- motion references
- asset style references
- opposite-category inspiration

## Smart Move: Look Outside the Category

For a dental clinic, study:
- luxury hospitality
- spa/editorial wellness
- premium architecture
- calm product photography

For construction SaaS, study:
- logistics control rooms
- architectural blueprints
- industrial design
- data visualization

For AI product, study:
- editorial science graphics
- information architecture
- calm productivity tools
- not only AI gradient sites

## Reference Extraction Template

```md
Reference:
Type: competitor / visual / motion / typography / layout / opposite-category

Borrow:
- ...

Avoid:
- ...

Transform:
- ...

Risk:
- ...

Could become:
- ...
```

## Do Not Copy

Never copy:
- exact layout
- exact animation
- exact copy
- distinctive brand device
- proprietary art
- unique illustration style too closely

Copy the principle, not the artifact.

## Moodboard to Design Translation

Moodboard item:
- "quiet mineral photography"

Translate into:
- off-white / stone palette
- soft shadows
- large whitespace
- close-up detail images
- slow fades
- low contrast borders
- ceramic material motif

Moodboard item:
- "technical control room"

Translate into:
- dark graphite UI
- grid lines
- precise labels
- status indicators
- monospace metadata
- directional motion

## Reference Anti-Pattern

Bad:
"Make it like Apple."

Better:
"Borrow Apple's restraint, product focus, and whitespace; avoid copying its layout, gradients, and exact typography rhythm. Differentiate with [business-specific metaphor]."
