# Starter Project Template Master

The active production starter is `starter-next-awwwards/`.

It exists so Claude Code does not rebuild the boring foundation every time.

## Starter responsibilities

- Next.js app structure
- global typography and design tokens
- reduced-motion provider
- hero system with CSS and R3F paths
- legal placeholder assets
- screenshot/QA scripts
- performance-aware Three.js defaults

## Claude Code workflow

If no project exists, run:

```bash
python ops/create_project.py --name "awwwards-site"
```

Then work inside:

```txt
workspace/awwwards-site/
```

## Rule

Do not let the starter control the concept. The starter is scaffolding; the concept must still come from research, strategy, and art direction.
