# Reference Mining

Build the moodboard that will discipline the concept competition. A small focused board beats a large unfocused one every time.

## First action

Read:

1. `.claude/skills/award-website-os/references/21-reference-mining-moodboard.md`
2. `.claude/skills/award-website-os/references/45-reference-mining-and-moodboards.md`
3. `.claude/skills/award-website-os/references/61-current-awwwards-benchmark-mining.md`
4. `.claude/skills/award-website-os/references/67-award-winning-case-study-patterns.md`

## Required output

`docs/benchmark-reference-board.md` with this structure:

```
## Category references (3–5)
For each: URL, why it won, signature idea, what to steal (technique), what to reject.

## Out-of-category references (3–5)
Same fields.

## Anti-references (3–5)
Generic sites in the same category. Why they fail.

## Specialist references (3 max)
One for typography, one for motion/3D, one for content/copy.

## Synthesis
Three sentences:
- What this category currently looks like.
- What it could look like.
- Where this project is positioned between those two.
```

## Mining rules

- Steal techniques, never surfaces.
- Cite the URL for every reference. No "general inspiration".
- Reject anything that can't survive being explained out loud in one sentence.
- More than 20 references means the concept isn't ready; cut.
- Anti-references are mandatory — without them the concept will drift toward the category average.

## What to look for in references

| Aspect | What to extract |
|---|---|
| Typography | Pairing, scale, rhythm, line breaks |
| Color | Role of color as an idea, not as decoration |
| Composition | Asymmetry, density, negative space rules |
| Motion | Easing vocabulary, choreography, restraint |
| 3D / WebGL | Whether 3D earned its place; the fallback strategy |
| Copy | Voice consistency, specificity, structure |
| Mobile | Whether mobile is a distinct design |

## Synthesis test

After the board is built, write the synthesis paragraph and read it aloud. If it could describe most sites in the category, the board is not differentiated enough yet. Push further.
