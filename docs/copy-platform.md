## Brand Idea

**The drop floor you can trust.**

Fitsole should speak like a sharp retail editor: direct, product-literate, confident, and useful. The copy should build desire while removing doubt.

## Voice

Use:
- Short sentences.
- Concrete product language.
- Retail confidence.
- Clear action labels.
- Specific trust details.

Avoid:
- "Innovative", "next-generation", "unlock", "revolutionary".
- Fake luxury language.
- Empty sneaker hype.
- Unsupported scarcity.
- Claims that sound like legal guarantees unless approved.

## 5-Second Message

What is this?
- A sneaker and fashion ecommerce store.

Who is it for?
- Shoppers looking for sneakers, apparel, kids, sale, and trusted local buying paths.

Why care?
- Products, prices, sizes, delivery, exchange, and support are made clear.

What next?
- Shop new drops, shop sale, search, or browse categories.

## Homepage Copy Direction

Hero headline options:
- New drops. Checked and ready.
- The drop floor is open.
- Find the pair. Check the fit. Move.

Hero support copy:
- Shop current sneakers, apparel, and sale picks with clear sizes, payment options, and support before checkout.

Primary CTAs:
- Shop new drops.
- Shop sneakers.
- Shop sale.

Trust strip labels:
- Authenticity process.
- Local delivery.
- Easy exchange.
- Payment options.
- Support that answers.

Use owner-approved details only. If a claim is not verified, soften it or replace it with a policy link.

## Product Card Microcopy

Labels:
- New.
- Sale.
- Sold out.
- Low stock only if inventory data supports it.
- Fitsole pick only if owner confirms curated merchandising.

Actions:
- View product.
- Choose size.
- Add to cart.
- Notify me only if back-in-stock app exists.

Do not hide price language behind icons.

## PDP Copy Direction

Buying panel:
- Size.
- Size guide.
- Add to cart.
- Delivery and exchange.
- Payment options.
- Need help? Contact support.

Product details:
- Use bullets for material, fit/use case, color, model, care, and included items only when verified.
- Avoid generic "premium quality" without product-specific evidence.

Trust copy:
- "Authenticity process" is safer than "100% authentic" unless owner legally approves exact claim.
- Delivery windows must match policy.
- Return/exchange language must match policy.

## Cart And Checkout Copy

Cart:
- Your cart.
- Size.
- Quantity.
- Subtotal.
- Continue shopping.
- Checkout.

Errors:
- Choose a size before adding.
- This size is currently unavailable.
- We could not update the cart. Try again or contact support.

Rules:
- Checkout CTA should never be clever.
- Do not use hype copy in error states.
- Payment/shipping caveats must be clear.

## SEO Copy Rules

- Collection headings should use real category terms: sneakers, shoes, apparel, kids, sale, brand names if authorized.
- Avoid keyword stuffing.
- Product names should match actual product catalog.
- Meta descriptions should mention product category, price/sale, delivery/support only if accurate.
- Do not create pages around unsupported claims like "official" or "licensed" without owner/legal confirmation.

## Claim Approval List

Requires owner confirmation before use:
- Authenticity wording.
- Licensed retailer wording.
- Branch count.
- Delivery windows and regions.
- Return/refund/exchange rules.
- BNPL/payment partner names.
- Warranty or guarantee language.
- Low-stock or bestseller labels.
- Staff pick/curated labels.

## Copy QA Checklist

- Can the hero pass the 5-second test?
- Does every section have a shopping job?
- Is every trust claim legally approved?
- Are CTAs direct and commerce-safe?
- Is mobile copy shorter than desktop copy where needed?
- Did we remove generic tech and fake luxury language?
