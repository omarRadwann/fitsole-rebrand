# 04 — Image Generation and Asset Sourcing Master

## Mission

Give the website premium visual assets legally and truthfully.

## Asset Decision Tree

For each section, choose:

1. Real user/client asset
2. Licensed stock photo/video
3. Generated AI image
4. Custom SVG/illustration
5. Canvas/WebGL procedural visual
6. Blender/3D asset
7. Typography-only composition

## When to Use Generated Images

Use generated images for:
- abstract brand worlds
- mood images
- editorial hero visuals
- non-realistic scenes
- symbolic visuals
- backgrounds/textures
- product visualization drafts clearly not real
- style exploration

Avoid generated images for:
- real team members
- real patients/customers
- legal/medical proof
- fake before/after
- exact product claims
- recognizable copyrighted characters/brands
- celebrity/person likeness without rights

## Example Prompt — Luxury Dental Clinic

```txt
Create a premium editorial hero image for a luxury dental clinic. 
Scene: abstract calm treatment studio, soft mineral-white surfaces, translucent glass, gentle daylight, precise clinical details without showing a real patient. 
Mood: calm, trustworthy, high-end, not cold. 
Composition: wide hero crop, large negative space for headline on left, subtle depth on right. 
Materials: ceramic, frosted glass, brushed metal, soft fabric. 
Avoid: smiling stock photo, visible medical procedure, fake doctor, exaggerated perfect teeth, blue corporate clinic look.
```

## Example Prompt — Construction SaaS

```txt
Create an abstract visual system for a construction operations SaaS. 
Scene: dark graphite blueprint grid transforming into live glowing site-status layers. 
Mood: technical, controlled, premium, practical. 
Elements: route lines, task nodes, material blocks, timeline layers. 
Composition: cinematic hero background with clear space for text. 
Avoid: generic hard hats, cranes, fake dashboards, random neon blob.
```

## Stock Search Query Templates

Use:
- `[business type] editorial interior premium`
- `[industry] detail photography material process`
- `[service] authentic workplace candid`
- `[product category] studio macro material`
- `[architecture/construction/dental/etc] premium detail no people`

## Licensing Checklist

For every external asset:
- source URL
- license
- commercial use allowed?
- attribution required?
- modification allowed?
- model/property release?
- AI training/generation restrictions?
- risk notes

## Optimization Checklist

Images:
- convert to AVIF/WebP
- responsive sizes
- compress
- set width/height
- lazy-load below fold
- preload critical hero only

Video:
- poster
- compressed WebM/MP4
- muted
- playsinline
- no autoplay sound
- pause control if meaningful

SVG:
- optimize
- remove junk metadata
- add title/aria if meaningful
