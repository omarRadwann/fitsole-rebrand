# 19 — Asset Generation Prompts Library

Use these as starting points. Always adapt to business, audience, and concept.

## Prompt Skeleton

```txt
Create [asset type] for [business/site].
Purpose: [hero/section/background/texture/OG].
Audience emotion: [trust/luxury/calm/energy].
Visual world: [metaphor/material].
Composition: [layout/crop/negative space].
Lighting: [soft daylight/cinematic/high contrast].
Palette: [tokens/mood].
Style: [editorial/product/abstract/documentary].
Avoid: [clichés, fake proof, copyrighted style, misleading people].
Aspect ratio: [16:9/4:5/1:1/9:16].
Usage: [web hero/background/card/OG].
```

## Luxury Service Hero

```txt
Create a premium editorial hero visual for a luxury [service type].
No fake customers or staff. 
Scene: abstract, calm, material-rich environment that suggests [service value] without literal stock-photo clichés.
Materials: [ceramic / glass / stone / fabric / brushed metal].
Lighting: soft directional light, high-end editorial atmosphere.
Composition: wide crop, strong negative space for headline, one memorable object/detail.
Avoid: smiling stock people, fake awards, black-gold cliché, over-polished plastic look.
```

## SaaS Mechanism Visual

```txt
Create an abstract product-mechanism visual for a [SaaS category].
Show [chaos/pain] transforming into [clarity/outcome].
Elements: nodes, paths, layers, dashboard fragments, process states, but no fake readable customer data.
Mood: precise, trustworthy, technical, premium.
Composition: cinematic hero background with a clear focal point.
Avoid: generic purple gradient, fake analytics dashboard, meaningless floating cards.
```

## Construction / Architecture Visual

```txt
Create a premium visual for [construction/architecture business].
Theme: material honesty, precision, planning becoming reality.
Elements: blueprint grid, material samples, light, concrete/steel/wood texture, measured lines.
Mood: confident, crafted, operational, not generic.
Composition: editorial spread, strong shadows, room for text.
Avoid: generic hard hats, cranes at sunset, stock handshake, fake project photo.
```

## Healthcare / Clinic Visual

```txt
Create a calm, trustworthy visual for a healthcare/clinic website.
Scene: abstract treatment environment or detail-focused studio image.
Mood: human, clean, warm, precise.
Lighting: soft daylight, calm shadows.
Materials: frosted glass, ceramic, clean fabric, gentle metal.
Avoid: fake doctors, fake patients, surgical procedure, exaggerated smiles, cold corporate blue cliché.
```

## Abstract Texture Prompt

```txt
Create a seamless abstract texture for a premium website background.
Material: [paper/mineral/glass/noise/fabric/metal].
Mood: subtle, tactile, quiet.
Usage: web background overlay.
Must be low contrast and not distract from text.
Avoid obvious patterns, logos, text, people, high noise, harsh colors.
```

## OG Image Prompt

```txt
Create an Open Graph image for [brand/business].
Include no small unreadable text unless provided as exact brand title.
Visual: [same motif as website].
Composition: bold central visual, safe margins, high contrast.
Mood: memorable and premium.
Aspect ratio: 1200x630.
Avoid fake claims, fake logos, crowded layout.
```

## Negative Prompt Library

Use as applicable:
- no fake people
- no fake customer logos
- no readable fake data
- no copyrighted characters
- no brand logos unless provided
- no overused gradient blobs
- no cliché stock photo
- no extra fingers/faces
- no text artifacts
- no watermark
- no clinical gore
- no exaggerated luxury gold
- no random futuristic holograms
