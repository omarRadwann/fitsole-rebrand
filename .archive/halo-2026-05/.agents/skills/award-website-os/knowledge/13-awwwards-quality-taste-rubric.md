# 13 — Awwwards-Level Taste Rubric

Use this to judge whether the site feels award-capable, not merely "nice."

## 10-Point Taste Test

Score each from 1–10.

### 1. Immediate Identity
Can a user tell what this brand/site is within 5 seconds?

Low: mystery with no payoff.
High: curiosity plus orientation.

### 2. Visual Ownership
Could this visual system belong to another random business?

Low: template.
High: tied to category, concept, and audience.

### 3. Typography Confidence
Does the type carry the design before images/effects?

Low: default sizes/weights.
High: intentional rhythm, line breaks, scale, tracking.

### 4. Composition
Does the layout feel composed or assembled?

Low: repeated centered sections.
High: hierarchy, asymmetry, negative space, rhythm.

### 5. Interaction Meaning
Do interactions explain/reward/guide?

Low: decorative motion.
High: concept-specific interaction.

### 6. Content Specificity
Could the copy be pasted onto a competitor?

Low: "innovative solutions."
High: specific pain, mechanism, result.

### 7. Mobile Integrity
Does mobile feel designed?

Low: compressed desktop.
High: own composition and interaction model.

### 8. Performance Feel
Does the site feel light and responsive?

Low: delayed, janky, heavy.
High: instant clarity, smooth polish.

### 9. Detail Consistency
Are small details part of the same world?

Low: mixed radius, random icons, inconsistent shadows.
High: tokens, motifs, states, microcopy aligned.

### 10. Memory
What will the user remember?

Low: nothing.
High: one sentence + one visual/interaction.

## Award Readiness Threshold

Do not call it award-level unless:
- average score ≥ 8.5
- no category below 7.5
- Memory ≥ 8.5
- Mobile Integrity ≥ 8
- Performance Feel ≥ 8
- Content Specificity ≥ 8

## Brutal Review Questions

Ask:
1. Would I screenshot this?
2. Would I send this to a designer friend?
3. Is the hero stronger without the copy? If not, typography/composition may be weak.
4. Is the copy stronger without the visuals? If not, messaging may be weak.
5. Is there one decision that feels brave?
6. Is there one decision that feels restrained?
7. Did we choose something instead of choosing everything?
8. Would this still work if WebGL failed?
9. Would this still work on a 390px phone?
10. Does every section earn its place?

## Taste Failure Modes

### "AI Fancy"
Lots of gradients, glass, shadows, animated cards, but no idea.

Fix:
- remove 50%
- define metaphor
- redesign hero
- rewrite copy

### "Dribbble Landing"
Pretty blocks, no business conversion.

Fix:
- add proof
- clarify CTA
- show mechanism
- answer objections

### "Awwwards Imitation"
Overanimated, confusing, copy hidden.

Fix:
- restore navigation
- simplify scroll
- clarify above fold
- add accessible DOM content

### "Developer Demo"
Cool WebGL, weak typography/content.

Fix:
- art direction pass
- copy pass
- responsive composition
- proof sections

### "Template Premium"
Clean but forgettable.

Fix:
- stronger concept
- custom hero object
- signature interaction
- category-specific visual motif
