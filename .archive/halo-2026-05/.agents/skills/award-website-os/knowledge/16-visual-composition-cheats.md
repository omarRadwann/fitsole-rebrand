# 16 — Visual Composition Cheats

## Hero Composition Patterns That Feel Premium

### 1. Editorial Split
Left: precise copy.
Right: strong object/image.
Use for SaaS, services, architecture, healthcare.

Smart trick:
- let the object slightly invade the text grid
- add small metadata labels
- avoid equal 50/50 symmetry unless luxury calm is intended

### 2. Oversized Type + Tiny Proof
Huge headline with small precise proof/control elements.

Works when:
- typography is strong
- copy is short
- imagery is subtle

### 3. Object on Grid
A 3D/visual object sits on a technical grid or material surface.

Works for:
- construction
- logistics
- AI
- finance
- hardware
- engineering

### 4. Cinematic Crop
Use a close crop of material/process instead of full stock scene.

Works for:
- luxury
- architecture
- healthcare
- product
- food

### 5. Calm Center With Edge Detail
Centered hero can work if edge details create depth:
- side annotations
- subtle grid
- off-canvas object
- ambient motion
- ticker/proof strip

## Section Rhythm

Avoid:
- hero
- centered text
- 3 cards
- centered text
- 3 cards
- CTA

Better rhythm:
1. strong hero
2. thin proof strip
3. asymmetric problem section
4. sticky mechanism
5. editorial case/proof
6. compact FAQ
7. cinematic CTA

## Spacing Tricks

- Increase top/bottom section padding more than default.
- Use fewer max-widths.
- Align small labels to big type edges.
- Let one element break the grid per section.
- Use whitespace as luxury, but not emptiness.
- Give mobile sections breathing room, not just smaller font.

## Typography Tricks

- Manually control hero line breaks.
- Tighten display tracking.
- Use small uppercase labels sparingly.
- Use numeric stats in tabular/mono style.
- Use contrast between large emotional headline and precise technical caption.
- Avoid mixing too many weights.
- Use body copy line length around 55–75 characters.

## Card Tricks

If cards are necessary:
- vary card sizes
- make one dominant
- include real specifics
- add micro data
- make hover state concept-specific
- avoid same icon-title-text pattern everywhere

## Image Tricks

- Crop details instead of full scenes.
- Use masks tied to concept.
- Use image captions as proof.
- Use image sequences for process.
- Add material closeups.
- Avoid smiling stock person unless truly authentic.

## Mobile Composition

Mobile should not be "desktop stacked."

Smart mobile alternatives:
- hero object behind cropped headline
- sticky bottom CTA
- horizontal proof chips
- tap-to-reveal feature chapters
- simplified static poster for 3D
- shorter headline with same meaning
- large touch targets
