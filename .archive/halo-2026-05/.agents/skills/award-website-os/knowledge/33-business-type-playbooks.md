# 33 — Business Type Playbooks

## Local Service

Default conversion:
- call
- WhatsApp
- book appointment
- request quote

Must include:
- location
- service area
- trust proof
- process
- FAQ
- contact visibility
- mobile sticky CTA if appropriate

## SaaS

Default conversion:
- book demo
- start trial
- see product

Must include:
- pain
- mechanism
- workflow demo
- use cases
- proof
- integrations if relevant
- security/trust if B2B

## Portfolio / Agency

Default conversion:
- view work
- contact
- book call

Must include:
- point of view
- selected projects
- case studies
- process
- capabilities
- contact

## E-commerce

Default conversion:
- buy
- add to cart
- subscribe

Must include:
- product clarity
- variants
- price/shipping
- trust/returns
- reviews if real
- hosted checkout
