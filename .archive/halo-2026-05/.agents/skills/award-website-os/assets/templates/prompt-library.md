## Smart Visual Polish

Do a senior art-direction polish pass.
Improve typography, composition, spacing, visual motif, mobile layout, and detail consistency.

## 3D Justification Gate

Before adding 3D, explain why 3D is better than image/SVG/CSS.
Define performance budget, fallback, mobile behavior, and reduced-motion behavior.
If it is decorative, reject it.

screenshot critic, accessibility UX, GPU performance, release QA, and orchestrator.
